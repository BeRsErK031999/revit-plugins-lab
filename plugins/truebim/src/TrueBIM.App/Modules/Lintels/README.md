# Перемычки

Модуль автоматизации оформления перемычек для спецификаций.

Первый этап намеренно не изменяет модель Revit. Сейчас модуль предоставляет кнопку во вкладке `КР`, безопасную команду-статус и изолированную точку расширения `TrueBIM.App.Modules.Lintels`.

Дальнейшая реализация начинается с read-only диагностики родительских перемычек и вложенных компонентов на рабочем RVT-файле. Полная последовательность, критерии готовности и открытые вопросы находятся в [`docs/Lintels/ROADMAP.md`](../../../../../../docs/Lintels/ROADMAP.md).
