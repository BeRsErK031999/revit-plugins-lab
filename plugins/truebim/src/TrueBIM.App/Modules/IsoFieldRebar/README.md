# Армирование по изополям

Изолированный модуль TrueBIM для подготовки армирования стен и плит по распознанным зонам изополей.

## Текущий безопасный MVP

- Последовательное WPF-окно `источник → зоны → host → правила → сравнение → применение` с живым индикатором готовности.
- Загрузка готового JSON-контракта `schemaVersion: "1.0"` и экранный preview контуров.
- Встроенное offline-распознавание цветовой шкалы, числовых диапазонов `см²/м`, подписей `d10s200+...` и hotspot-зон PNG; внешний CLI worker остаётся опциональным override.
- Таблица ручной коррекции зон: исключение шумовых контуров, смена диапазона по легенде и объединение зон одного слоя/класса в общий convex hull.
- Комплект `As1X/As2X/As3Y/As4Y` с проверкой ролей по имени и заголовку, manifest, SHA-256 и явным назначением верх/низ.
- Управляемые `DetailCurve`-линии preview на активном 2D-виде с явной очисткой.
- Выбор стены или горизонтальной плиты; для плиты — привязка по двум опорным и одной проверочной точке, локальные оси и итоговый overlay.
- Managed-отсечение зон по внешнему контуру и отверстиям плиты с поддержкой нескольких регионов и контролем сохранённой площади.
- Сохранение профиля `документ + вид + плита`; при загрузке профиль всегда повторно проверяется на текущих зонах.
- Блокировка расчёта правил при вырожденной привязке, отклонении третьей точки более 50 мм, точках вне верхней грани или полностью потерянной зоне.
- Инженерные правила плиты по верхней границе диапазона `см²/м`: разбор `d10s200+...`, формула площади и безопасная проверка `принято >= требуется`.
- Явные режимы `только дополнительное усиление поверх базовой сетки` и `полное сочетание внутри зон`.
- Раскладка линий по clipped-регионам и отверстиям с защитным слоем, граничным отступом, минимальной длиной, X/Y и верх/низ.
- Экранный preview количества и линий стержней до записи; строгий выбор `RebarBarType` по диаметру и стабильная маркировка `layer/face/zone id`.
- Diff повторного запуска `добавить / обновить / удалить / без изменений` по выбранной плите; сравнение фактической геометрии и диаметра со стабильной сигнатурой.
- Отдельная таблица проверки до записи с поиском и фильтрами по слою, статусу, диаметру, шагу и confidence; устаревший diff автоматически блокирует применение.
- Ручная настройка строки перед сравнением: выбор распознанного или пользовательского сочетания, исключение зоны, проверка площади и сброс одной либо всех настроек с пересчётом preview.
- Обратимое объединение касающихся инженерных зон с одинаковыми правилами: polygon union без convex hull, мультивыбор `Ctrl/Shift` и стабильный id независимо от порядка выделения.
- Парный JSON/CSV-отчёт по зонам и итогам слоёв: SHA-256 исходников, worker/plugin version, привязка, инженерные настройки, rule fingerprint и текущее состояние diff.
- QA геометрии до записи: блокирующие дефицит площади, пересечения зон одного слоя и выход итоговых контуров за host; предупреждения о покрытии, clipping и исходных зонах вне host принимаются пользователем только для текущего preview.
- Отчёт schema `1.1` сохраняет QA fingerprint, покрытие каждого слоя, классифицированные ошибки/предупреждения и факт их принятия.
- Идемпотентное применение одной транзакцией: изменяются только принадлежащие модулю элементы с `TrueBIM IsoFieldRebar; id=...`, ручная арматура не затрагивается.
- Раздельные placement services для простых горизонтальных плит и прямых стен.
- Логирование, sample JSON и unit-тесты для воспроизводимого QA.

## Важное ограничение

P4.1 создаёт отдельные стержни по инженерным правилам и отсечённым регионам, но это ещё не финальный выпускной режим. `Полное сочетание` действует только внутри распознанных зон и не достраивает фоновую сетку всей плиты; результат не группируется в `Rebar Set`/`Area Reinforcement`, нет нахлёстов и анкеровки. P5.3b предотвращает дубли и поддерживает ручную конфигурацию зон. P6.2 добавляет воспроизводимый отчёт и геометрический QA, но короткий итог после создания со ссылкой на последний отчёт остаётся следующим срезом.

Полный план до рабочего производственного сценария: [`docs/IsoFieldRebar/ROADMAP.md`](../../../../../../docs/IsoFieldRebar/ROADMAP.md).
