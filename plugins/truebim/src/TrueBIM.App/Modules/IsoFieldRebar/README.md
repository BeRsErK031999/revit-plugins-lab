# Армирование по изополям

Изолированный модуль TrueBIM для подготовки армирования стен и плит по распознанным зонам изополей.

## Текущий безопасный MVP

- Последовательное WPF-окно `источник → зоны → host → правила → создание` с живым индикатором готовности.
- Загрузка готового JSON-контракта `schemaVersion: "1.0"` и экранный preview контуров.
- Встроенное offline-распознавание цветовой шкалы и hotspot-зон PNG; внешний CLI worker остаётся опциональным override.
- Комплект `As1X/As2X/As3Y/As4Y` с проверкой ролей по имени и заголовку, manifest, SHA-256 и явным назначением верх/низ.
- Управляемые `DetailCurve`-линии preview на активном 2D-виде с явной очисткой.
- Выбор стены или плиты, ручная калибровка и read-only диагностика правил.
- Контролируемая запись по одному пробному `Rebar` на валидную зону после подтверждения.
- Раздельные placement services для простых горизонтальных плит и прямых стен.
- Логирование, sample JSON и unit-тесты для воспроизводимого QA.

## Важное ограничение

Текущий write-flow создаёт пробные одиночные стержни. Это не производственная раскладка: встроенный режим пока строит консервативные convex-hull зоны без чтения числовых значений шкалы; также нет привязки к реальному контуру и отверстиям плиты, защитного слоя, полноценных наборов, идемпотентного обновления и инженерного отчёта.

Полный план до рабочего производственного сценария: [`docs/IsoFieldRebar/ROADMAP.md`](../../../../../../docs/IsoFieldRebar/ROADMAP.md).
