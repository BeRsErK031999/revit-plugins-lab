# Армирование по изополям

Экспериментальный модуль для будущего распознавания изополей и подготовки правил армирования стен и плит в Revit.

## Уже реализовано

- Изолированный namespace `TrueBIM.App.Modules.IsoFieldRebar`.
- Минимальные DTO для точки, полилинии, результата распознавания и правила армирования.
- Интерфейс `IIsoFieldRecognitionRunner`.
- Безопасная заглушка `StubIsoFieldRecognitionRunner`, которая не подключает OpenCV, Python или ML-зависимости.
- Кнопка `Армирование по изополям` в панели `БИМ`, открывающая WPF-окно модуля.
- Выбор изображения или JSON-файла изополей через отдельный file picker service.
- Кнопка проверки stub-runner без реального распознавания и без изменений модели Revit.
- JSON-контракт `schemaVersion: "1.0"` для результата распознавания.
- Reader JSON-файлов изополей с базовой валидацией обязательных полей.
- Экранный preview JSON-контуров в WPF-окне без Revit `Transaction` и без создания элементов.
- Управляемые линии предпросмотра на активном 2D-виде Revit через `DetailCurve` с явной очисткой.
- Выбор стены или плиты как будущего host-элемента без изменения модели Revit.
- Калибровка координат изображения через image anchor, масштаб `мм/пикс` и инверсию оси Y для Revit preview.
- Read-only preview правил армирования для распознанных зон с валидацией host, типа арматуры и шага.
- Controlled write-flow для создания одной тестовой арматуры на выбранном host-элементе только после явного подтверждения пользователя.
- Slab-specific placement для простых плит: валидные зоны превращаются в параллельные тестовые Rebar-линии внутри bounding box плиты.

## Пока не реализовано

- Реальное распознавание контуров.
- Полноценная wall-specific раскладка для стен.
- Учет отверстий, наклонных плит и compound structure.
