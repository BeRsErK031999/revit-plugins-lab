# Печать

Модуль пакетной печати и экспорта листов Revit в PDF, DWG и DXF.

Текущий этап: кнопка, команда, manifest, подключение к TrueBIM launcher, MVP-окно выбора листов активного документа, конструктор имени файла и прямой экспорт отдельных PDF, DWG и DXF. Выбор export setups и объединенный PDF добавляются следующими шагами по плану в `docs/print-module-plan.md`.
