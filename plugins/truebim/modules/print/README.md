# Печать

Модуль пакетной печати и экспорта листов Revit в PDF, DWG и DXF.

Текущий этап: кнопка, команда, manifest, подключение к TrueBIM launcher, MVP-окно выбора листов активного документа, конструктор имени файла, прямой экспорт отдельных PDF, DWG и DXF, а также выбор DWG/DXF export setup из сохраненных настроек Revit.

Для DWG и DXF окно показывает отдельные настройки экспорта. Если в документе нет сохраненных CAD export setups или пользователь оставляет значение по умолчанию, модуль экспортирует через стандартные `DWGExportOptions` / `DXFExportOptions` и отображает fallback в статусе.

Объединенный PDF и расширенные параметры качества экспорта добавляются следующими шагами по плану в `docs/print-module-plan.md`.
