# Свернуть ВРС

Модуль создает копию выбранной спецификации Revit и скрывает в копии числовые столбцы, где все значения равны нулю.

## Рабочий сценарий

1. Открыть проект Revit.
2. Открыть TrueBIM и запустить `Свернуть ВРС`.
3. В окне выбора указать спецификацию, которую нужно свернуть.
4. Получить копию спецификации с суффиксом `- свернутая`.

## Правила безопасности

- Исходная спецификация не изменяется.
- Дубликат создается и настраивается в одной транзакции Revit.
- Поля, скрытые в исходной спецификации, остаются скрытыми в копии.
- Итоговые колонки остаются видимыми.
- Текстовые и служебные колонки остаются видимыми.
- Скрываются только числовые колонки, где все найденные числовые значения равны нулю.

## План развития

- Manual QA на реальной ВРС из проекта заказчика.
- Автоматическая замена размещенной на листе спецификации на свернутую копию, если это подтвердит рабочий сценарий.
