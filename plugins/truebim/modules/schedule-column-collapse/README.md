# Свернуть ВРС

Модуль изменяет выбранную спецификацию Revit и скрывает числовые столбцы, где все значения равны нулю.

## Рабочий сценарий

1. Если исходную спецификацию нужно сохранить без изменений, заранее сделать ее копию в Revit.
2. Открыть TrueBIM и запустить `Свернуть ВРС`.
3. В окне выбора источника указать один из вариантов: активное окно, выбранная спецификация в диспетчере проекта или выбор из списка с поиском.
4. Если выбран список, указать спецификацию, которую нужно свернуть.
5. Получить ту же спецификацию со скрытыми нулевыми столбцами.

## Правила безопасности

- Выбранная спецификация изменяется напрямую.
- Изменения выполняются в одной транзакции Revit.
- Поля, скрытые до запуска команды, остаются скрытыми.
- Текстовые и служебные колонки остаются видимыми.
- Итоговые числовые колонки скрываются, если все их значения равны нулю, и остаются видимыми при наличии ненулевых значений.
- Числовые заголовки вроде `-10` и `-12` не считаются значениями столбца.

## План развития

- Manual QA на реальной ВРС из проекта заказчика.
- Автоматический предпросмотр списка столбцов, которые будут скрыты, если это потребуется для согласования.
