# Свернуть ВРС

Модуль создаёт копию выбранной или активной спецификации Revit и скрывает в копии числовые столбцы, где все значения равны нулю.

## Рабочий сценарий

1. Открыть спецификацию или лист со спецификацией.
2. Если на листе несколько спецификаций, выбрать нужную спецификацию.
3. Открыть TrueBIM и запустить `Свернуть ВРС`.
4. Получить копию спецификации с суффиксом `- свернутая`.

## Правила безопасности

- Исходная спецификация не изменяется.
- Дубликат создаётся и настраивается в одной транзакции Revit.
- Поля, скрытые в исходной спецификации, остаются скрытыми в копии.
- Итоговые колонки остаются видимыми.
- Текстовые и служебные колонки остаются видимыми.
- Скрываются только числовые колонки, где все найденные числовые значения равны нулю.

## План развития

- Manual QA на реальной ВРС из проекта заказчика.
- Опция выбора нескольких спецификаций на листе.
- Автоматическая замена размещённой на листе спецификации на свернутую копию, если это подтвердит рабочий сценарий.
