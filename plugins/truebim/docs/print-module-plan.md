# Модуль "Печать"

Status updated: 2026-07-17.

Пользовательские замечания от 2026-07-16, их диагностика по логу и последовательность исправлений вынесены в [`print-feedback-roadmap.md`](print-feedback-roadmap.md). Первый этап — быстрый сбор метаданных без предварительной графической обработки всех листов — выполнен в коде и ожидает ручной проверки в Revit.

## Статус релиза

Первый рабочий релиз модуля `truebim.print` остается release candidate для практической QA и локального использования в Revit 2022 и Revit 2025. Пользовательская проверка выявила отдельный RoadMap исправлений, поэтому стабильный релиз не фиксируется до прохождения сценариев из `print-feedback-roadmap.md`.

Считается завершенным для первого релиза:

- кнопка `Печать` на панели `БИМ`, команда Revit, manifest `truebim.print` и подключение к launcher;
- окно выбора печатаемых листов из открытых документов Revit;
- сохранение выбора листов при переключении фильтра источника;
- конструктор имени файла с предпросмотром, нормализацией и базовыми токенами;
- прямой экспорт PDF отдельными файлами и одним объединенным PDF на документ-источник;
- симметричные флаги `PDF + Один PDF` и `DWG + Один DWG` без конкурирующего списка PDF-режимов;
- PDF-настройки цвета, качества растровых элементов и raster/vector режима;
- прямой экспорт DWG/DXF;
- фактическая печать выбранных листов через установленный Windows-принтер и `PrintManager.SubmitPrint(View)`;
- выбор текущего или именованного Revit print setup, предупреждение для PDF-драйверов и восстановление исходных настроек печати;
- выбор DWG/DXF export setup из сохраненных настроек Revit и fallback на настройки по умолчанию;
- сохранение базовых настроек окна в `%APPDATA%\TrueBIM\<RevitVersion>\print-settings.json`;
- единое окно `Печать и экспорт` с явными режимами, выбором PDF/DWG/DXF/DWF и раскрываемыми подробными настройками;
- именованные локальные пресеты заказчиков в `%APPDATA%\TrueBIM\<RevitVersion>\print-presets.json`;
- unit-тесты чистой логики печати и прохождение Release build/test;
- локальный deploy и installer artifacts через preflight для Revit 2022 и Revit 2025.

Остается за рамками первого релиза:

- вкладки по источникам вместо текущего фильтра;
- чтение и экспорт связанных моделей;
- print sets;
- фильтры и группировка по параметрам листов, альбомам или сборкам;
- расширенные CAD-параметры сверх сохраненных Revit export setups.

## Цель

Создать модуль TrueBIM для пакетной печати и экспорта листов Revit в PDF, DWG и DXF с удобным выбором листов, конструктором имени файлов и поддержкой нескольких источников листов.

Ориентир по пользовательскому сценарию - удобство уровня ModPlus "Экспорт листов": таблица листов, настройки чтения, выбор форматов, маски имени, сохраненные настройки, открытые документы и связанные модели.

## Объем первого релиза

1. Кнопка `Печать` на панели `БИМ`.
2. Модуль `truebim.print` в реестре TrueBIM и launcher.
3. Окно печати с выбором листов из открытых документов Revit.
4. Экспорт выбранных листов в PDF, DWG и DXF.
5. Отдельные PDF и объединенный PDF.
6. Выбор DWG/DXF export setup из сохраненных настроек Revit.
7. Конструктор имени файла с предпросмотром.
8. Сохранение базовых настроек модуля.
9. Unit-тесты чистой логики, local deploy и installer preflight.

## Архитектура

### Команда и модуль

- `OpenPrintCommand` - внешняя команда Revit.
- `TrueBimCommandActions.OpenPrint` - единая точка запуска из ribbon и launcher.
- `PrintModule` - описание модуля для `ModuleRegistry`.
- `plugins/truebim/modules/print/module.json` - installed manifest.

### UI

Окно модуля должно состоять из четырех основных зон:

- настройки чтения листов;
- таблица листов;
- панель настроек экспорта;
- строка состояния и прогресс.

В таблице листов нужны колонки:

- выбран;
- источник;
- номер листа;
- имя листа;
- формат;
- PDF/DWG/DXF статус;
- предпросмотр имени файла.

### Экспорт

PDF:

- для Revit 2022 и 2025 использовать прямой `Document.Export` с `PDFExportOptions`;
- поддержать отдельные PDF и объединенный PDF;
- для объединенного PDF использовать один вызов `Document.Export` со списком выбранных листов, `Combine = true` и общим именем файла;
- добавить настройки PDF: цвет, качество растровых элементов и raster/vector режим;
- управляемый пакетный PDF создавать прямым экспортом; PDF-принтеры допускаются только в режиме фактической печати с явным предупреждением о собственном диалоге и пути драйвера.

DWG/DXF:

- использовать `Document.Export` с `DWGExportOptions` и `DXFExportOptions`;
- брать существующие export setups из документа через `BaseExportOptions.GetPredefinedSetupNames`;
- при выборе setup использовать `DWGExportOptions.GetPredefinedOptions` или `DXFExportOptions.GetPredefinedOptions`;
- для DWG применять профиль TrueBIM поверх базового Revit setup: `FileVersion`, `Colors`, `PropOverrides`, `TargetUnit`, `SharedCoords`, `ExportingAreas`, `MergedViews`, hide-флаги, `ExportOfSolids`, `LineScaling`, `TextTreatment`, layer/linetype/pattern files и non-plot/hatch background опции, если они доступны в текущей версии Revit API;
- если setup не выбран или сохраненных настроек нет, использовать настройки Revit по умолчанию и показывать это в статусе;
- не мутировать глобальные Revit Export Setup без явного действия пользователя; отдельная кнопка создает новый `ExportDWGSettings` из текущего профиля.

### Конструктор имени

Базовая маска:

```text
{Номер листа}_{Имя листа}
```

Нужные токены:

- `{Номер листа}` (`{SheetNumber}` для старых настроек);
- `{Имя листа}` (`{SheetName}` для старых настроек);
- `{Номер проекта}` (`{ProjectNumber}` для старых настроек);
- `{Имя проекта}` (`{ProjectName}` для старых настроек);
- `{Имя документа}` (`{DocumentName}` для старых настроек);
- `{Дата:yyyy-MM-dd}` (`{Date:yyyy-MM-dd}` для старых настроек);
- `{Счетчик}` и `{Счетчик:000}` (`{Counter}` / `{Counter:000}` для старых настроек);
- `{Параметр листа:...}` и `{Параметр проекта:...}`.

Обязательные правила:

- чистить запрещенные символы Windows;
- предупреждать о дублях;
- показывать итоговый путь до запуска;
- ограничивать слишком длинные имена.

## План работ

Текущий статус:

- шаг 1 выполнен: каркас кнопки и модуля добавлен;
- шаг 2 выполнен: окно печати показывает листы активного документа, выбор листов, папку и форматы экспорта;
- шаг 3 выполнен: конструктор имени файла обновляет preview по маске и предупреждает о проблемах;
- шаг 4 выполнен для первого релиза: добавлен прямой экспорт отдельных и объединенных PDF с настройками цвета, качества растровых элементов и raster/vector режима;
- шаг 5 выполнен для первого релиза: добавлен прямой экспорт DWG/DXF, выбор сохраненных Revit export setups, расширенный DWG-профиль поверх `DWGExportOptions` и fallback на настройки по умолчанию;
- шаг 6 выполнен для первого релиза в варианте фильтра источника: окно печати собирает листы из нескольких открытых документов Revit, фильтрует таблицу по источнику, сохраняет выбор при переключении источников и экспортирует выбранные листы через документ-источник;
- шаг 7 выполнен для первого релиза: базовые настройки печати сохраняются в `%APPDATA%\TrueBIM\<RevitVersion>\print-settings.json` и восстанавливаются при следующем открытии окна;
- единый вход `Печать` и именованные пресеты заказчиков выполнены: PDF/DWG выбираются в одном окне, а полный снимок настроек сохраняется в `%APPDATA%\TrueBIM\<RevitVersion>\print-presets.json`;
- фактическая печать выполнена в коде: отдельный режим показывает установленные Windows-принтеры, Revit print setups и диапазон выбранных листов, отправляет каждый лист через `PrintManager.SubmitPrint(View)`, журналирует ошибки драйвера и восстанавливает исходные настройки печати;
- PDF/DWG переключатели унифицированы: формат включается первым флагом, объединение — вторым; legacy-настройка `отдельные PDF и один общий` мигрирует в `Один PDF` без ошибки чтения пресета;
- опция `Открыть папку` сохраняется в последнем состоянии и пресетах; после всего пакетного экспорта папка открывается один раз только при наличии фактически созданных файлов;
- шаг 8 выполнен для первого релиза: чистая логика покрыта unit-тестами, Release build/test прошли, local deploy и installer artifacts проверены preflight для Revit 2022 и Revit 2025;
- следующий шаг вне первого релиза: вкладки по источникам, linked models, print sets и расширенные фильтры.
