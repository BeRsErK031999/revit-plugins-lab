# Модуль "Печать"

## Цель

Создать модуль TrueBIM для пакетной печати и экспорта листов Revit в PDF, DWG и DXF с удобным выбором листов, конструктором имени файлов и поддержкой нескольких источников листов.

Ориентир по пользовательскому сценарию - удобство уровня ModPlus "Экспорт листов": таблица листов, настройки чтения, выбор форматов, маски имени, сохраненные настройки, открытые документы и связанные модели.

## Объем первого релиза

1. Кнопка `Печать` на панели `БИМ`.
2. Модуль `truebim.print` в реестре TrueBIM и launcher.
3. Окно печати с выбором листов активного документа.
4. Экспорт выбранных листов в PDF, DWG и DXF.
5. Конструктор имени файла с предпросмотром.
6. Сохранение базовых настроек модуля.

## Архитектура

### Команда и модуль

- `OpenPrintCommand` - внешняя команда Revit.
- `TrueBimCommandActions.OpenPrint` - единая точка запуска из ribbon и launcher.
- `PrintModule` - описание модуля для `ModuleRegistry`.
- `plugins/truebim/modules/print/module.json` - installed manifest.

### UI

Окно модуля должно состоять из четырех основных зон:

- настройки чтения листов;
- таблица листов;
- панель настроек экспорта;
- строка состояния и прогресс.

В таблице листов нужны колонки:

- выбран;
- источник;
- номер листа;
- имя листа;
- формат;
- PDF/DWG/DXF статус;
- предпросмотр имени файла.

### Экспорт

PDF:

- для Revit 2022 и 2025 использовать прямой `Document.Export` с `PDFExportOptions`;
- поддержать отдельные PDF и объединенный PDF;
- на первом этапе не использовать виртуальные PDF-принтеры.

DWG/DXF:

- использовать `Document.Export` с `DWGExportOptions` и `DXFExportOptions`;
- брать существующие export setups из документа;
- показывать ошибку, если нужный setup отсутствует или лист не экспортируемый.

### Конструктор имени

Базовая маска:

```text
{SheetNumber}_{SheetName}
```

Нужные токены:

- `{SheetNumber}`;
- `{SheetName}`;
- `{ProjectNumber}`;
- `{ProjectName}`;
- `{DocumentName}`;
- `{Date:yyyy-MM-dd}`;
- `{Counter}`;
- параметры листа и project information.

Обязательные правила:

- чистить запрещенные символы Windows;
- предупреждать о дублях;
- показывать итоговый путь до запуска;
- ограничивать слишком длинные имена.
