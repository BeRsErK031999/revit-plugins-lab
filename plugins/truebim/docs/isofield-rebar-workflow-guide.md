# Методичка: армирование по изополям

![Армирование по изополям: порядок работы](assets/isofield-rebar-workflow-card.svg)

Эта методичка относится только к текущему каркасу модуля `Армирование по изополям`. Она описывает безопасный порядок проверки JSON/изображения изополей, preview, выбора host-элемента и создания тестовой арматуры.

## Перед запуском

1. Откройте тестовую модель Revit со стеной или плитой.
2. Перейдите на обычный 2D-вид, где можно показать preview-линии.
3. Убедитесь, что в модели есть доступные типы арматуры `RebarBarType`.
4. Держите рядом лог `%APPDATA%\TrueBIM\Logs\truebim.log`.

## Основной порядок

1. Откройте вкладку `TrueBIM`.
2. На панели `БИМ` нажмите `Армирование по изополям`.
3. Выберите JSON fixture из `docs/IsoFieldRebar/examples/` или изображение изополей.
4. Нажмите распознавание или загрузку файла и проверьте, что контуры появились в окне модуля.
5. Нажмите `Показать в Revit` только на активном 2D-виде.
6. Проверьте, что preview-линии появились и удаляются через `Очистить`.
7. Выберите host: простую прямую стену или прямоугольную плиту.
8. Нажмите расчет правил и проверьте read-only список правил без изменения модели.
9. Нажмите создание тестовой арматуры только после явного подтверждения.
10. Проверьте созданный тестовый элемент и сразу выполните Revit Undo.

## Что считается корректным результатом

- JSON `schemaVersion: "1.0"` читается без ошибки.
- Контуры видны в окне модуля.
- Preview в Revit создается только как временные управляемые линии и очищается.
- Правила армирования показываются до записи в модель.
- Тестовая арматура создается только после подтверждения.
- Один Revit Undo возвращает модель в исходное состояние.

## Ограничения текущего каркаса

- Реальное распознавание контуров остается внешним worker-ом.
- Запись в модель сейчас создает тестовую арматуру, а не финальную раскладку.
- Стены поддерживаются в простом прямом сценарии.
- Плиты проверяются по bounding box без учета отверстий, уклонов и сложной структуры.

## Что приложить к отчету об ошибке

- версию Revit;
- входной JSON или изображение;
- тип host-элемента: стена или плита;
- скриншот окна `Армирование по изополям`;
- лог `%APPDATA%\TrueBIM\Logs\truebim.log`;
- описание шага: загрузка файла, preview, выбор host, расчет правил или тестовая запись.

После сборки установщика этот файл должен лежать в установленном пакете по пути `%APPDATA%\TrueBIM\<RevitVersion>\Docs\isofield-rebar-workflow-guide.md`.
