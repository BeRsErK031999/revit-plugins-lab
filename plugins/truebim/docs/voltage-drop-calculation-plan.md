# Расчет потери напряжения: план переноса из Excel

Дата анализа: 2026-07-03.

Источник: `docs/Расчет_потери_напряжения_в_кабеле,_освещенности,_КЗ,_заземления.xlsm`.

## Первый лист Excel

Лист: `Расчет потери напряжения`.

Диапазон листа: `A1:BD104`.

На листе есть 57 формульных ячеек и 14 вставленных изображений со справочными таблицами. Формулы и кэшированные значения читались напрямую из OpenXML-пакета `.xlsm`, без изменения книги.

В книге также есть листы:

- `Годовой расчет потребления`
- `Расчет освещенности `
- `Распределение по фазам`
- `Расчет токов КЗ`
- `Расчет заземляющего устройства `
- `Заполняемость кабельных лотков `

Текущий перенос ограничен первым листом, как указано в задаче.

## Таблицы первого листа

### Таблица 1. Исходные данные и расчет потери напряжения

Входные и справочные ячейки:

- `B2 = 44`: коэффициент C для Al, 400 В
- `B3 = 72.2`: коэффициент C для Cu, 400 В
- `B4 = 12.1`: коэффициент C для Cu, 230 В
- `B5 = 7.7`: коэффициент C для Al, 230 В
- `B6 = 310`: длина линии L
- `B7 = 150`: сечение кабеля F
- `B8 = 40`: мощность Pp

Формулы:

- `B9 = B6*B8`
- `B11 = B9/(B2*B7)`
- `B12 = B9/(B3*B7)`
- `B13 = B9/(B4*B7)`
- `B14 = B9/(B5*B7)`
- `B16:B22 = B8/(0.38*1.73*cos)`
- `B24:B30 = B8/(0.22*cos)`

### Таблица 2. Квартиры с электроплитами и лифты

Диапазон: строки `63:68`.

Основные входы:

- `B66 = 7`: количество этажей
- `C67 = 25`: количество квартир
- `F67 = 10`: Ру., кВт на квартиру
- `I67 = 1`, `J67 = 0.9`, `L67 = 0.98`: коэффициенты и cos для квартир
- `D68 = 2`: количество лифтов
- `H68 = 30`: Ру.общ. лифтов
- `J68 = 1`, `L68 = 0.8`: коэффициент и cos для лифтов

Формулы:

- `G67`: интерполяция `Ко/Руд` по количеству квартир
- `H67 = C67*F67`
- `K67 = C67*G67*I67*J67`
- `M67 = SQRT(N67*N67-K67*K67)`
- `N67 = K67/L67`
- `O67 = K67/(0.38*SQRT(3)*L67)`
- `I68`: коэффициент спроса лифтов по этажности и количеству лифтов
- `K68 = H68*I68*J68`
- `M68 = SQRT(N68*N68-K68*K68)`
- `N68 = K68/L68`
- `O68 = K68/(0.38*SQRT(3)*L68)`

### Таблица 3. Квартиры повышенной комфортности и лифты

Диапазон: строки `70:77`.

Основные входы:

- `B73 = 8`: количество этажей
- `C73 = 7`: количество квартир
- `F73 = 11.5`: Ру., кВт на квартиру
- `C74 = 24`: количество квартир для коэффициента `G74`
- `I73 = 1`, `J73 = 1`, `L73 = 0.98`: коэффициенты и cos для квартир
- `D75 = 2`: количество лифтов
- `H75 = 30`: Ру.общ. лифтов
- `J75 = 1`, `L75 = 0.8`: коэффициент и cos для лифтов
- `L77 = 0.98`: cos общего расчета по квартирам

Формулы:

- `G73`: интерполяция коэффициента по `F73`
- `G74`: интерполяция коэффициента по `C74`
- `H73 = C73*F73`
- `K73 = C73*F73*G73*G74*I73*J73`
- `M73 = SQRT(N73*N73-K73*K73)`
- `N73 = K73/L73`
- `O73 = K73/(0.38*SQRT(3)*L73)`
- `I75`: коэффициент спроса лифтов по этажности и количеству лифтов
- `K75 = H75*I75*J75`
- `M75 = SQRT(N75*N75-K75*K75)`
- `N75 = K75/L75`
- `O75 = K75/(0.38*SQRT(3)*L75)`
- `K77 = K67+K73`
- `M77 = SQRT(N77*N77-K77*K77)`
- `N77 = K77/L77`
- `O77 = K77/(0.38*SQRT(3)*L77)`

Замечание: формула `G73` в Excel содержит ветку `F73-400` для диапазона `40..49`. В первом переносе логика сохранена как в Excel, но это нужно отдельно подтвердить как норму или ошибку в исходном файле.

- `image1:image4`: допустимые токовые нагрузки для внутренней прокладки кабелей
- `image3:image4`: допустимые токовые нагрузки для кабелей в земле

В первом срезе они используются как источник формул и справочных значений. Для полноценного справочника в Revit их нужно оцифровать в структурированные таблицы, чтобы пользователь мог выбирать материал, напряжение, сечение, способ прокладки и коэффициенты без ручного переноса из картинки.

## План работ

1. Добавить первый вертикальный срез в Revit: кнопку `Расчет потери напряжения` на панель `ЭОМ`, модальное окно, три расчетные таблицы, подсветку вводимых полей, live-пересчет и тесты на значения из Excel.
2. Добавить строгую валидацию диапазонов и понятные сообщения для значений вне справочных интервалов.
3. Оцифровать справочные изображения первого листа в модели данных: допустимые токовые нагрузки, коэффициенты спроса, коэффициенты C.
4. Связать справочные таблицы с UI: выпадающие списки для материала кабеля, напряжения, типа прокладки, сечения и категории потребителя.
5. Расширить перенос на дополнительные формульные блоки первого листа в строках `75:104`, если они должны входить в этот же модуль.
6. Провести ручную QA-проверку в Revit 2025: наличие кнопки в `ЭОМ`, открытие модального окна, пересчет при изменении входов, читаемость русских подписей.

## Первый срез реализации

В первый срез включены:

- новая кнопка Ribbon на панели `ЭОМ`
- команда открытия модального окна
- три расчетные таблицы из первого листа
- подсветка полей ввода зеленым цветом
- расчетный сервис без зависимости от Revit API
- unit-тесты на совпадение с кэшированными значениями Excel

## Второй срез реализации

Второй срез добавляет строгую валидацию входных диапазонов:

- коэффициенты C и сечение кабеля должны быть больше 0
- длина линии и мощность не могут быть отрицательными
- количество этажей, квартир и лифтов проверяется как целое число
- количество квартир для справочных коэффициентов ограничено диапазоном `1..1000`
- Ру. для квартир повышенной комфортности ограничено диапазоном `1..70 кВт`
- коэффициенты спроса проверяются как `0..1`
- `cos(φ)` проверяется как `(0..1]`

При ошибке модальное окно подсвечивает конкретные поля красным и показывает текст причины в статусной строке. Расчетный слой больше не возвращает тихий `NaN` для неподдержанного справочного диапазона.
