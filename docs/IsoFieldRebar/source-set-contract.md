# IsoField source set contract

Версия контракта: `1.0`.

Расчётные изображения загружаются одним комплектом из четырёх файлов. Один файл соответствует одному расчётному слою; перед запуском worker плагин проверяет комплект целиком.

## Обязательные роли

| Роль | Направление | Грань плиты | Статус назначения |
| --- | --- | --- | --- |
| `As1X` | X | требует подтверждения | определяется по имени файла, доступна ручная коррекция |
| `As2X` | X | требует подтверждения | определяется по имени файла, доступна ручная коррекция |
| `As3Y` | Y | требует подтверждения | определяется по имени файла, доступна ручная коррекция |
| `As4Y` | Y | требует подтверждения | определяется по имени файла, доступна ручная коррекция |

До подтверждения соответствия `As1...As4` верхней/нижней грани интерфейс не должен подменять расчётную роль инженерной интерпретацией.

## Автоматическое назначение

- Роль определяется без учёта регистра по вхождению `As1X`, `As2X`, `As3Y` или `As4Y` в имя файла.
- Имя вида `...As1X1.png` корректно определяется как `As1X`.
- Если роль не найдена или имя содержит более одной роли, файл остаётся неназначенным.
- Пользователь может изменить роль каждого файла в выпадающем списке.

## Блокирующая валидация

Комплект готов, только если одновременно выполнены условия:

- выбрано ровно четыре разных файла;
- каждая обязательная роль назначена ровно один раз;
- каждый файл существует и читается как изображение;
- ширина и высота всех изображений совпадают.

Разный pixel size считается признаком несовместимого экспорта или масштаба. Геометрическая проверка масштаба по контрольным точкам относится к этапу P3.

## Обработка

Настроенный CLI worker вызывается последовательно в порядке `As1X`, `As2X`, `As3Y`, `As4Y`. Результаты объединяются в один `IsoFieldRecognitionResult`:

- `id` контура получает префикс роли, например `As1X:zone-a`;
- `layerRole` заполняется ролью исходного файла;
- диагностика получает префикс `[As1X]` и не теряет связь с источником.

JSON-контракт зон остаётся отдельным совместимым входом и выбирается без изображений.
